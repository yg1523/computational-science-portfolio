# Computational Science Portfolio

This repository contains selected technical materials accompanying my ICME application.

Contents include:
- Computational PDE coursework on diffusion, elliptic flow, multigrid, wave propagation, and PML
- Computational Linear Algebra coursework on weighted least squares, QR factorisation, GMRES, preconditioning, and eigenvalue algorithms
- Dissertation work on variational analysis of the Euler equation on a rotating sphere

The emphasis throughout is on numerical stability, solver design, spectral analysis, and modelling.
